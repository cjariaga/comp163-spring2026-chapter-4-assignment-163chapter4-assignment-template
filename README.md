# College Life Adventure Game

**Course:** COMP 163 - Introduction to Programming  
**Assignment:** Chapter 4 - Branching  
**Author:** Clayan Ariaga  
**GitHub:** cjariaga

## Description
A text-based game that simulates a college semester. Make choices about course load, study focus, and manage your GPA, study hours, social life, and stress levels.

## How to Run
```bash
python assignment.py
```

## Gameplay
1. Enter your starting stats (GPA, study hours, social points, stress level)
2. Choose your course load (Light, Standard, or Heavy)
3. Pick a subject to focus on (Programming, Math, English, History)
4. See your final semester results!

## Features
- Multiple branching paths based on your choices
- Different endings depending on your final stats
- Input validation with 'quit' option
- Uses Chapter 4 concepts: if-else, logical operators, membership operators, nested conditionals

## Requirements
- Python 3.x
- Chapters 1-4 programming concepts

Type 'quit' at any input prompt to exit the game.
